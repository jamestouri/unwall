# Contributing to unwall

Thanks for your interest in contributing.

## Prerequisites

- Node.js >= 18
- npm

## Setup

```bash
git clone https://github.com/unwall-xyz/unwall.git
cd unwall
npm install
npm run build
```

## Running Tests

```bash
npm test
```

## Submitting Changes

1. Fork the repo and create a feature branch
2. Keep changes focused — one feature or fix per PR
3. Add tests for new functionality
4. Run `npm run build` and `npm test` before submitting
5. Write a clear PR description explaining what and why

## Reporting Issues

Open an issue at [github.com/unwall-xyz/unwall/issues](https://github.com/unwall-xyz/unwall/issues).
